## Linearizing content on tinymobile

<div class="flex flex-spacebetween mb2 onmobile-flex-column">
  <div class="w49 flex-self-vcenter">
	Applies <code>float: none; display: block; width: auto;</code> to the element for tinymobile breakpoint.
  </div>
  <div class="w49">
<Source
  language='html'
  dark
  format

`}

  </div>

<br />

<strong>Flexbox version:</strong> resize the window, by default, elements take all the width on mobile thanks to <code>ontinymobile-flex-column</code>.

<div class="flex flex-spacebetween mb2 ontinymobile-flex-column">
  <div class="w49 flex-self-vcenter">

      <div class="bg-pm-blue-light p1 w30">.w30</div>
      <span class="bg-pm-blue p1 w30">.w30</span>
      <em class="bg-pm-blue-dark p1 w30">.w30</em>
    </div>
  </div>
  <div class="w49 flex-self-vcenter">
<Source
  language='html'
  dark
  format
  code={`
<div className="flex flex-spacebetween ontinymobile-flex-column">
  <div className="w30">div</div>
  <span className="w30">span</span>
  <em className="w30">em</em>
</div>
`}
/>

  </div>
</div>

Other flexbox helpers for tinymobile:

- <code>ontinymobile-flex-self-start</code> applies <code>align-self: start</code> to the element for mobile and smaller breakpoints.



## Margin/padding helpers

Please check before how <a href="?path=/docs/front-end-helpers-margin-padding--page">margin/padding helpers</a> work.

- <code>.ontinymobile-m</code> = <code>margin</code>
- <code>t/r/b/l</code> = <code>top/right/bottom/left</code>
- <code>0/1</code> = 0/1em

Examples:

<div class="flex flex-spacebetween mb2 onmobile-flex-column">
  <div class="w49 onmobile-mb1 bordered-container">
    <div class="ontinymobile-pl1 ontinymobile-pr1 ontinymobile-m1 bg-pm-blue-light">
      <code>.ontinymobile-m1</code>  = <code>margin: 1em</code> <span class="color-white">on tinymobile</span>.
    </div>
  </div>
</div>
