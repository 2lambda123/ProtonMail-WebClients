## Absolute and other helpers

<div class="flex flex-spacebetween mb2 onmobile-flex-column">
  <div class="w49 flex-self-vcenter">
    Applies <code>position: absolute</code> to element + <code>z-index: 1</code>.
  </div>
  <div class="w49">
    <pre class=" language-markup prismjs"><code>&lt;span <strong>class="absolute"</strong>&gt;…&lt;/span&gt;
</code></pre>
  </div>
</div>

<br /><br />

<div class="flex flex-spacebetween mb2 onmobile-flex-column">
  <div class="w49 flex-self-vcenter">
    Centers absolute-positionned element via <code>top: 50%; left: 50%; transform: translate(-50%, -50%);</code>.
  </div>
  <div class="w49">
    <pre class=" language-markup prismjs"><code>&lt;span <strong>class="centered-absolute"</strong>&gt;…&lt;/span&gt;
</code></pre>
  </div>
</div>

<br /><br />

<div class="flex flex-spacebetween mb2 onmobile-flex-column">
  <div class="w49 flex-self-vcenter">
    Centers absolute-positionned element via <code>left: 50%; transform: translate(-50%, 0);</code>.
  </div>
  <div class="w49">
    <pre class=" language-markup prismjs"><code>&lt;span <strong>class="centered-absolute-horizontal"</strong>&gt;…&lt;/span&gt;
</code></pre>
  </div>
</div>

<br /><br />

<div class="flex flex-spacebetween mb2 onmobile-flex-column">
  <div class="w49 flex-self-vcenter">
    <code>covered-absolute</code> class applies <code>position: absolute; top: 0; right: 0; bottom: 0; left: 0;</code>.
  </div>
  <div class="w49">
    <pre class=" language-markup prismjs"><code>&lt;span <strong>class="covered-absolute"</strong>&gt;…&lt;/span&gt;
</code></pre>
  </div>
</div>

<br /><br />

<div class="flex flex-spacebetween mb2 onmobile-flex-column">
  <div class="w49 flex-self-vcenter">
    Applies <code>top: 0; right: 0;</code> to absolute-positionned element.
  </div>
  <div class="w49">
    <pre class=" language-markup prismjs"><code>&lt;span <strong>class="top-right"</strong>&gt;…&lt;/span&gt;
</code></pre>
  </div>
</div>

<br /><br />

<div class="flex flex-spacebetween mb2 onmobile-flex-column">
	<div class="w49 flex-self-vcenter">
	  Applies <code>bottom: 0;</code> to absolute-positionned element.
	</div>
  <div class="w49">
	  <pre class=" language-markup prismjs"><code>&lt;span <strong>class="bottom"</strong>&gt;…&lt;/span&gt;
  </code></pre>
	</div>
</div>

In case of, there are helpers on mobile like <code>onmobile-static</code> which can cancel fixed/absolute positionning.
