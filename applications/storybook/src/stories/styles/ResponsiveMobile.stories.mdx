  <div class="w49 flex-self-vcenter">
<Source
  language='html'

  </div>

This can also be used on <code>.autogrid</code> class.

Other flexbox helpers for mobile:

- <code>onmobile-flex-item-fluid</code> applies <code>flex: 1 1 0px</code> to the element for mobile and smaller breakpoints.
- <code>onmobile-flex-item-fluid-auto</code> applies <code>flex: 1 1 auto</code> to the element for mobile and smaller breakpoints.
- <code>onmobile-no-flex</code> applies <code>flex: 0 0 auto</code> to the element for mobile and smaller breakpoints.
- <code>onmobile-flex-shrink</code> applies <code>flex-shrink: 1</code> to the element for mobile and smaller breakpoints.
- <code>onmobile-flex-self-start</code> applies <code>align-self: start</code> to the element for mobile and smaller breakpoints.
- <code>onmobile-flex-wrap</code> applies <code>flex-wrap: wrap;</code> to the element for mobile and smaller breakpoints.

## Margin/padding helpers

Please check before how <a href="?path=/docs/front-end-helpers-margin-padding--page">margin/padding helpers</a> work.

- <code>.onmobile-m</code> = <code>margin</code>
- <code>.onmobile-p</code> = <code>padding</code>
- <code>t/r/b/l</code> = <code>top/right/bottom/left</code>
- <code>0/0.25/0.5/1/2/3.5</code> = 0/0.25em/0.5em/1em/2em/3.5em

Examples:

<div class="flex flex-spacebetween mb1 onmobile-flex-column">
  <div class="w49 onmobile-mb1 bordered-container">
    <div class="onmobile-pt2 bg-pm-blue-light ">
      <code>.onmobile-pt2</code> = <code>padding-top: 2em</code> <span class="color-white">on mobile and smaller</span>.
    </div>
  </div>
  <div class="w49 onmobile-mb1 bordered-container">
    <div class="onmobile-pl2 bg-pm-blue-light">
      <code>.onmobile-pl2</code> = <code>padding-left: 2em</code> <span class="color-white">on mobile and smaller</span>.
    </div>
  </div>
</div>
<div class="flex flex-spacebetween mb2 onmobile-flex-column">
  <div class="w49 onmobile-mb1 bordered-container">
    <div class="onmobile-p1 bg-pm-blue-light">
      <code>.onmobile-p1</code> = <code>padding: 1em</code> <span class="color-white">on mobile and smaller</span>.
    </div>
  </div>
  <div class="w49 onmobile-mb1 bordered-container">
    <div class="onmobile-pl1 onmobile-pr1 onmobile-m1 bg-pm-blue-light">
      <code>.onmobile-pl1</code> + <code>.onmobile-pr1</code> + <code>.onmobile-m1</code>  = <code>padding-left: 1em</code> + <code>padding-right: 1em</code> + <code>margin: 1em</code> <span class="color-white">on mobile and smaller</span>.
    </div>
  </div>
</div>

## Width/max-width helpers

<div><ul class="list-2columns onmobile-list-1column">
  <li><code>.onmobile-w25</code> = <code>width: 25%;</code></li>
  <li><code>.onmobile-w100</code> = <code>width: 100%;</code></li>
  <li><code>.onmobile-wauto</code> = <code>width: auto;</code></li>
  <li><code>.onmobile-mw100</code> = <code>max-width: 100%;</code></li>
  <li><code>.onmobile-min-w100</code> = <code>min-width: 100%;</code></li>
</ul></div>

These apply for mobile breakpoint and smaller.
