# Hiding/disabling elements


<div class="flex flex-spacebetween mb2 onmobile-flex-column">
  <div class="w49 flex-self-vcenter">
    Applies a <code>display: none</code> to element.
  </div>
  <div class="w49">
<Source
  language='html'
  dark
  format
  code={`
<span hidden>Lorem ipsum dolor</span>
<span className="hidden">Lorem ipsum dolor</span>
`}
/>
  </div>
</div>

<br /><br />

<div class="flex flex-spacebetween mb2 onmobile-flex-column">
  <div class="w49 flex-self-vcenter">
    Applies a <code>visibility: hidden</code> to element (keeps space used but hides it).
  </div>
  <div class="w49">
<Source
  language='html'
  dark
  format
  code={`
<span className="nonvisible">Lorem ipsum dolor</span>
`}
/>
  </div>
</div>

<br /><br />

<div class="flex flex-spacebetween mb2 onmobile-flex-column">
  <div class="w49 flex-self-vcenter">
    Makes element disappear but element remain vocalized by speech synthesis (<code>display: none</code>/<code>visibility: hidden</code> elements won't be vocalized)
  </div>
  <div class="w49">
<Source
  language='html'
  dark
  format
  code={`
<span className="sr-only">Lorem ipsum dolor</span>
`}
/>
  </div>
</div>

<br /><br />

<div class="flex flex-spacebetween mb2 onmobile-flex-column">
  <div class="w49 flex-self-vcenter">

    If you need to remove pointer events on all children of an element, you may use <code>no-pointer-events-children</code> class.
  </div>
  <div class="w49">
<Source
  language='html'
  dark
  format
  code={`
<a className="no-pointer-events">Lorem ipsum dolor</a>
`}
/>
  </div>
</div>
